# インストールガイド

## 前提条件

- PHP 8.2以上
- Composer
- Webサーバー (Apache または Nginx)

## Accela Wikiのインストール

リポジトリをクローンして、依存関係をインストールします。

```bash
git clone https://github.com/accela-framework/wiki.git
cd wiki
composer install
```

## Webサーバの設定

プロジェクトディレクトリをApacheのドキュメントルートとして設定してください。<br>
サブディレクトリで運用する場合は、Accelaの仕様に合わせてapp/内、もしくは.htaccessを修正する必要があります。<br>
nginxの場合は、適宜ルーティング処理を追加してください。

## 環境設定

環境設定ファイルを用意します。

```bash
mv env-sample.php env.php
```

`env.php`を編集して設定を変更することができます。
データディレクトリの変更が必要な場合は、ここから行います。

## ページを作成

クローン直後は`data/`ディレクトリにサンプルのドキュメントが含まれていますので、ファイル構成やフロントマターの書き方などの参考にしてください。

独自のコンテンツに置き換える場合は、`data/`内のファイルを編集・削除してください。
最低限必要なのは`data/index.md`（トップページ）のみです。

環境設定でデータディレクトリを変更した場合は、そのディレクトリを手動で作成してください。

## 確認

ブラウザでアクセスし、Wikiのトップページが表示されれば成功です。
