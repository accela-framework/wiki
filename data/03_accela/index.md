# Accelaについて

このWikiシステムで使用している**Accela**は、PHPで超高速なSPAが開発できるWebフレームワークです。<br>
依存関係もなく、Nodeや特殊な環境も必要ありません。格安のレンタルサーバでも動作します。

[Accela PHP Web Framework](https://accela.in-green-spot.com)
